# Dataset para Curie (localizable)

Ruta recomendada para trabajar en clase:

- `07-practica-guiada/data/olist/`

Descarga el dataset desde Kaggle:

- https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Coloca dentro de `07-practica-guiada/data/olist/` estos archivos:

- `olist_customers_dataset.csv`
- `olist_orders_dataset.csv`
- `olist_order_items_dataset.csv`
- `olist_order_payments_dataset.csv`
- `olist_products_dataset.csv`

Nota:

- Si no tienes estos CSV, usa el modo `simulado` del notebook `eda_curie_guiado.ipynb`.
